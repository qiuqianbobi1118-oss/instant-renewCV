# Safety and Privacy

This workflow is designed for public or semi-public resume and portfolio generation.

## Never Commit

- Personal phone numbers.
- Personal email addresses.
- Home addresses.
- Internal company screenshots.
- Proprietary table names.
- Server names.
- Private player data.
- Raw interview notes with identifiable people.
- Unreleased product information.

## Use Placeholders

Use placeholders in templates:

- `{{candidate_name}}`
- `{{email}}`
- `{{phone}}`
- `{{company}}`
- `{{project_name}}`
- `{{metric}}`

## External Design References

Do:

- Extract high-level design tokens.
- Learn layout rhythm.
- Study section hierarchy.
- Use public reference links.

Do not:

- Copy images.
- Copy proprietary CSS.
- Copy brand assets.
- Recreate another site so closely that it becomes confusingly similar.
