# Copywriting Principles

The portfolio page should not read like a pasted resume.

## Write Like This

- Start with what the candidate noticed.
- Explain why the problem mattered.
- Show the decision logic.
- Show what was built or changed.
- Explain why it transfers to the target role.

## Avoid

- Responsible for...
- Participated in...
- Assisted with...
- Improved efficiency without evidence.
- Generic AI/product language.

## Better Pattern

```text
The team did not need another dashboard. It needed a way for non-data teammates to ask better questions without waiting for an analyst. I built the workflow around that constraint.
```
